Популярні дистрибутиви Linux

(Текст заголовків взято з Вікіпедії)

Debian

Debian — операційна система, що складається з вільного програмного забезпечення з відкритим кодом.

Ubuntu

Ubuntu — дистрибутив Linux, заснований на Debian. Розробляється компанією Canonical Ltd.

Arch Linux

Arch Linux — незалежний дистрибутив Linux, оптимізований для архітектури x86-64.

Fedora

Fedora — дистрибутив Linux, що розробляється спільнотою Fedora Project за підтримки Red Hat.
