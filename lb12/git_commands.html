Основні команди Git

Create a Repository

Створити новий репозиторій:

$ git init

Клонувати існуючий репозиторій:

$ git clone my_url

Observe your Repository

Статус файлів:

$ git status

Зміни у файлах (ще не додані):

$ git diff

Історія змін:

$ git log

Working with Branches

Список гілок:

$ git branch

Створити нову гілку:

$ git branch new_branch

Перейти на іншу гілку:

$ git checkout my_branch

Об'єднати гілки:

$ git merge branch_name

Synchronize & Changes

Додати файли до індексу:

$ git add .

Зробити коміт:

$ git commit -m "message"

Завантажити зміни з сервера:

$ git pull

Відправити зміни на сервер:

$ git push
